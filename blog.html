What is DOM?

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of an HTML or XML page as a tree of objects. Each element (like a paragraph, div, or button) becomes a node in this tree. JavaScript can use the DOM to read, modify, add, or remove elements on the page. For example, clicking a button can change text using DOM methods. In simple terms, the DOM lets you interact with and control web pages dynamically.

How do you select an element from DOM?

To select an element from the DOM, you can use JavaScript methods like:
1. getElementById(): Selects an element by its unique ID.
Example: document.getElementById("myButton")
2. getElementsByClassName(): Selects elements by class name.
Example: document.getElementsByClassName("myClass")
3. querySelector(): Selects the first matching element using CSS selectors.
Example: document.querySelector(".myClass")
4. querySelectorAll(): Selects all matching elements.
Example: document.querySelectorAll(".myClass")
These methods allow you to easily interact with elements in the DOM!

What is event delegation in the context of the DOM? and why it is a fault?

Event delegation in the DOM means assigning a single event listener to a parent element instead of individual child elements. The parent element listens for events that bubble up from child elements. This makes it more efficient because you donâ€™t need multiple listeners for each child. It is especially useful for handling dynamically added elements. However, it can be faulty if event propagation is not handled properly. Event bubbling can sometimes cause confusion in determining which element triggered the event.

How do you manupulate an elements attreabutes and styles using the DOM?

To manipulate an element's attributes, you can use methods like
1. getAttribute() to get an attribute's value.
Example: element.getAttribute("src");
2. setAttribute() to set a new attribute value.
Example: element.setAttribute("src", "newImage.jpg");
3. removeAttribute() to remove an attribute.
Example: element.removeAttribute("src");
To manipulate styles, use the style property for inline styles and classList for CSS classes.
